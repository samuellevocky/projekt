# 📚 Školská knižnica – PHP + MySQL

## 🧾 Popis projektu
Tento projekt je jednoduchá webová aplikácia – školská knižnica.  
Umožňuje správu kníh, používateľov a výpožičiek.

---

## ⚙️ Použité technológie
- PHP (bez frameworku)
- MySQL (mysqli)
- HTML + CSS
- JavaScript (dark mode toggle)

---

## 🗄️ Funkcie aplikácie

### 📌 Knihy
- pridávanie nových kníh (CREATE)
- zobrazenie kníh (READ)
- úprava kníh (UPDATE)
- mazanie kníh (DELETE)

### 👤 Používatelia
- evidencia používateľov

### 📖 Výpožičky
- požičanie knihy
- vrátenie knihy
- zobrazenie aktívnych výpožičiek

---

## 🌙 UI funkcie
- dark / light mode
- dashboard štatistiky
- moderné tabuľky
- animácie a hover efekty

---

## 🗄️ Databáza
Projekt používa MySQL databázu s tabuľkami:
- books
- users
- loans

---

## 🚀 Ako spustiť projekt
1. Importuj databázu (SQL súbor)
2. Nastav pripojenie v `db.php`
3. Spusť projekt cez XAMPP / Apache
4. Otvor `index.php` v prehliadači
